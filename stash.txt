Теперь вы хотите поменять ветку, но не хотите делать коммит с тем, над чем вы ещё работаете; 
тогда вы прячете эти изменения. Чтобы создать новую "заначку", выполните git stash:

$ git stash

Чтобы посмотреть, что у вас есть припрятанного, используйте git stash list:

$ git stash list

Чтобы вернуть припрятанное обратно:
 
$ git stash apply
